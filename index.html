智数营是一个培养数据科学家（Data Scientist）的黄埔军校，讲师是来自于国内统计分析领域、大数据领域的实战派，他们会把现实工作中应用到的技能手把手交给你，让你在新工作中快速到达工作层。另外智数营会将优秀的学员推荐给战略合作企业，一方面解决学员工作问题，另一方面节省企业人力培训成本。

初 级 课 程 系 列

软件工程

  • 版本管理工具: Git, SVN，HG
  • Bug跟踪工具: Bugzilla，FreeBug
  • 建模设计：VISIO
  • 进度管理工具：Project

Java基础

  • java基础介绍：包括：1.基本类型；2.包，类，方法；3.对象，String，Array
  • 开发工具介绍与编程实践：eclipse、Maven、Ant
  • java集合体系介绍

Linux系统使用

  • 基本的Shell命令
  • 管道和定向输出
  • MySQL简单使用

网络基础

  • HTTP, APIs, and Rest
  • Parsing
    • HTML and XML
    • JSON
    • PDF

大数据

  • Hadoop生态圈整体框架分布
  • HDFS、MapReduce原理操作
  • 数据挖掘（Java、Python）
  • 分布式数据存储(HBase（包含Zookeeper使用）)
  • 数据采集 (Flume, Scribe, Nginx）
  • 数据分析（Pig, Hive，Sqoop）
  • Hadoop集群部署管理

数据存储

  • NoSQL
    • 文档类(MongoDB)
    • 图类(Neo4j)
    • Key-Value (Redis, Memecache)
    • 列式存储（Cassandra）
  • 文件系统和文本

统计学与概率学

  • R语言基础
  • 数据抓取、数据预处理
  • 描述性统计 (mean, mode, variance, etc.)
  • 估计 (confidence intervals, sampling, etc.)
  • 相关性 (covariance, causation, etc.)
  • 分布
  • 意义 (Hypothesis testing, ANOVA, etc.)
  • 条件概率
  • 数据挖掘（R语言）

可视化

  • 交互性
    • Javascript
    • HTML
  • 地理和地图

机器学习

  • 无监督：聚类、关联分析、降维
  • 监督：分类、回归
  • 基于Hadoop的推荐：相似度指标、基于栏目 VS 基于用户 VS 基于内容、局限
  • 评价：Cross Validation, recall, precision, ROC
  • R语言分布式机器学习
  • Java语言分布式机器学习
